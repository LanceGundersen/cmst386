Popup

The final project will likely have a call-to-action in the form of a popup (modal) that announces a sale price or newsletter. These are often used to capture customers either via a direct sale or for follow up media/content. 

const modal = document.getElementById('myModal')
const span = document.getElementsByClassName('close')[0]

window.setTimeout(() => {
  modal.style.display = 'block'
}, 1000)

span.onclick = function() {
  modal.style.display = 'none'
}

SEO - Links

When building links into a website they should be used to provide amplifying information, navigate to subcategories, to a third party, etc. Depending on the type of website and the audience the types of links are important too such as opening a new tab or navigating the current tab. For example, a link that provides a free e-book download shouldn't replace the current website tab as the user may end up leaving the website early. Often these types of links are parts of the overall sales funnel or user experience.

Some general tips when building links are:

  1. A link should be able stand alone
  2. Links should be descriptive
  3. A link should start with a relevant keyword
  4. Links should be unique
  5. Plain text links should stick to the underlined text convention
  6. A link should be SEO friendly by being relevant to the target page
  7. A link shouldn't be overly keyword heavy

SEO - XML

Web search engines routinely go through and index all website pages they can access. One way they do this is by "poking" around the web and following pages. One way to think of this is through an analogy of a spider crawling through a maze. Sometimes if the spider doesn't understand the content/pages it visits it will fail to index it properly. For a website, this could mean it never reads your blog or the awesome video you've uploaded. One of the ways to work around this limitation of the search engines is to provide a map called a sitemap. This website has a sitemap.
